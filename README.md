## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

To run this project, you will need the following installed:
- Node.js (<a href="https://nodejs.org/en/download/package-manager/">Installation instructions</a>), and
- npm (<a href="https://www.npmjs.com/get-npm">Installation instructions</a>)

### Installing

First clone the project from GitHub
```sh
git clone https://github.com/lexlegis/BasicMathOperations.git
```
Then enter the project directory and install all dependencies
```sh
cd BasicMathOperations
npm install
```
Once the installation of dependencies is completed, run the application
```sh
node server.js
```
You should get a message like this:
```sh
Basic Math Operations app listening on port 8000
```
Then go to your browser, and go to the following URL:
```sh
localhost:8000
```
Application should be up and running, ready for use.

### Running the tests

You can run test simply by running
```sh
npm test
```
